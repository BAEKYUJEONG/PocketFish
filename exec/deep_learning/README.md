## CNN 모델 생성

- `cnn_model_structure` : 어종 분류 모델 구조 생성하는 파일
- `cnn_model_tester` : 분류 모델 결과 테스트하는 파일
- `flask_deploy` : 실제 서비스에서 사용하는 물고기 분류 api
